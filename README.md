# luisterspoor

Offline Dutch listening and recall practice.

Live app: https://evgeny.io/projects/luisterspoor/

This repository contains the static deployable player and final mixed audio tracks.

## Development

Serve locally from the repository root:

```bash
python3 -m http.server 8787
```

Then open http://127.0.0.1:8787/.

## Deployment

Merges to `main` deploy through GitHub Actions to `https://evgeny.io/projects/luisterspoor/`.
